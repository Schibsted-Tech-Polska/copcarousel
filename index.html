Lightweight carousel with touch support and no dependencies

This is the demo page.

If you're looking for some documentation, you should check out the project repo at GitHub (CopCarousel).

Available demos:

  1. Basic setup

  2. Filled with plug

  3. Single slide & string template

  4. Vertical

All demos use:
  • Data source: randomuser.me/
  • Common stylsheet with small modification per each: examples/common.css
  • Common JS with small modification per each: examples/common.js

Minimized versions:

In our team we use minimized versions of all our CopCarousel based carousels. To concat, minimize JS/CSS and then include in HTML we use GruntJS. You can check our script here: Gruntfile.js.

To run grunt you need NodeJS. Checkout the gh-pages branch of our repository. Then all you need to do is run these commands:

$ npm install -g grunt-cli
$ npm install
$ grunt
